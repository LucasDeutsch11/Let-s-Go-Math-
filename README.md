# Let's Go Math!

Lets Go Math! Is a simple math practice website that is built using Flask.

## Cycle 1 - Hello World Equivalency
This type of project demonstrates a working local Flask environment with:
-A homepage layout
-One math practice problem
-User input handling
-Correct/Incorrect feedback
